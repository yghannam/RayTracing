      Extend Assignment 1a to support a user defined camera and extend the kernel function to compute ray through every pixel and to include a function raySphere that computes and returns the nearest "t" of intersection of the ray where a sphere. Use the following data structures;

      • Camera: an object with components: viewpoint (origin), at point (at), up vector (up), field of view (fov)
      • Sphere: an object with components: center (C) and radius (R) (and some additional information).
      • Ray:  an object with components: origin (O) and direction (d)
